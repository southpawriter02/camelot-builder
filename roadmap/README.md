# Camelot Builder: A DAoC Realm Ability Calculator Roadmap

Welcome to the roadmap for **Camelot Builder**, an interactive, web-based Realm Ability (RA) calculator for the classic MMORPG, Dark Age of Camelot. This document outlines the planned features, enhancements, and technical considerations for building this tool.

The goal of Camelot Builder is to provide a fast, modern, and user-friendly platform for players to theorycraft character builds, share them with the community, and explore different specialization paths.

## Roadmap Structure

This roadmap is organized into several key areas, each representing a major component of the project. Each section is further broken down into specific, actionable feature documents.

### 1. [Core Functionality](./01-core-functionality/)
This section covers the essential features required to make the calculator functional and useful. It includes class selection, the mechanics of spending RA points, and the rules engine for build validation.

### 2. [Data Management](./02-data-management/)
Here, we detail the strategy for sourcing, structuring, and maintaining the vast amount of data related to classes, abilities, and game rules.

### 3. [UI/UX](./03-ui-ux/)
This section focuses on the user-facing aspects of the application, including the interactive interface, the build summary display, and ensuring a responsive, mobile-friendly design.

### 4. [Sharing and Persistence](./04-sharing-and-persistence/)
A key goal is to foster community engagement. This section outlines features like generating shareable links for builds and persisting a user's work across sessions.

### 5. [Advanced Features](./05-advanced-features/)
These are the stretch goals and long-term enhancements that will take Camelot Builder from a simple calculator to a comprehensive character planning tool. This includes user accounts, community-curated build lists, and integration with item databases.

## How to Contribute

This roadmap is a living document. Feedback, suggestions, and contributions are welcome. Please refer to the individual feature documents for detailed requirements and implementation ideas.
